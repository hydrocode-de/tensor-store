tensorage.backend.base¶

This module provides the base context class for working with tensors and datasets in Supabase.

It provides a BaseContext class that defines the interface for working with tensors and datasets in Supabase. This class is designed to be subclassed by specific implementations of the Supabase backend, such as the DatabaseContext class.
