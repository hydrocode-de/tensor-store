tensorage.store¶

This module defines the TensorStore class which is responsible for storing and retrieving tensor data from a Supabase backend. The store can be accessed by using Pythons set and get item methods. Additionally, all dataset keys are also available as attributes of the store object. The store supports numpy-style slicing and as of now accepts and returns numpy arrays.
