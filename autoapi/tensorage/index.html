tensorage.login(email: str | None = None, password: str | None = None, backend_url: str | None = None, backend_key: str | None = None) → tensorage.store.TensorStore¶

Log in to the Supabase backend using email and password authentication.

This function creates a BackendSession object using the provided backend URL and key, or the default values if none are provided. It then logs in to the backend session using the provided email and password. If the login is successful, it returns the tensor store instance for the backend session.

Parameters:

  • email – The email address of the user to log in.

  • password – The password of the user to log in.

  • backend_url – The URL of the Supabase backend. Defaults to None.

  • backend_key – The API key for the Supabase backend. Defaults to None.

Returns:

The tensor store instance for the backend session.

Raises:

RuntimeError – If the login fails.

tensorage.signup(email: str, password: str, backend_url: str | None = None, backend_key: str | None = None) → gotrue.types.AuthResponse¶

Sign up a new user to the Supabase backend using email and password authentication.

This function creates a BackendSession object using the provided backend URL and key, or the default values if none are provided. It then signs up a new user to the backend session using the provided email and password. If the signup is successful, it returns an AuthResponse object containing the user’s access token and refresh token.

Parameters:

  • email – The email address of the user to sign up.

  • password – The password of the user to sign up.

  • backend_url – The URL of the Supabase backend. Defaults to None.

  • backend_key – The API key for the Supabase backend. Defaults to None.

Returns:

An AuthResponse object containing the user’s access token and refresh token.

Raises:

RuntimeError – If the signup fails.
